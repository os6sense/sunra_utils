# SunraPS
A collection of utilities for process control.
ProcessWatch - A class which will run a block of code on process 
               termination.

## Installation

Add this line to your application's Gemfile:

    gem 'sunra_ps'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install sunra_ps
